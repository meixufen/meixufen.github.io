####添加文章方法

该语句就可以在_post文件夹中生成新的.md文件，该文件即博客的展示内容，文件语法遵循markdown语法。

编写完.md文档后重新使用hexo g和hexo d命令发布，即可看到刚刚撰写的blog。（可能不能即时生效，稍稍等那么一下，然后刷新页面，over）
